<!-- Abbreviations for automatic tooltips -->

*[URDF]: Unified Robot Description Format - XML format for robot kinematics and dynamics
*[SRDF]: Semantic Robot Description Format - Extends URDF with planning-specific information
*[FK]: Forward Kinematics - Computing end-effector pose from joint values
*[IK]: Inverse Kinematics - Computing joint values from end-effector pose
*[TCP]: Tool Center Point - The controlled point at the end-effector
*[DOF]: Degrees of Freedom - Number of independent motion axes
*[OMPL]: Open Motion Planning Library - Sampling-based motion planning algorithms
*[TrajOpt]: Trajectory Optimization - Optimization-based motion planning
*[SQP]: Sequential Quadratic Programming - Optimization algorithm for nonlinear problems
*[OSQP]: Operator Splitting Quadratic Program solver
*[QP]: Quadratic Programming - Optimization with quadratic objective
*[FCL]: Flexible Collision Library - Collision detection library
*[ACM]: Allowed Collision Matrix - Defines which link pairs to ignore in collision
*[LVS]: Longest Valid Segment - Interpolation-based continuous collision checking
*[RRT]: Rapidly-exploring Random Tree - Sampling-based planning algorithm
*[PRM]: Probabilistic Roadmap - Multi-query motion planning algorithm
*[OPW]: Opw Kinematics - Analytical IK for 6-DOF industrial robots
*[KDL]: Kinematics and Dynamics Library - Numerical kinematics solver
*[ABB]: ABB Ltd - Industrial robotics manufacturer
*[Hz]: Hertz - Cycles per second (planning frequency)
