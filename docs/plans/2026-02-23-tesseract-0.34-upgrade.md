### Task 4: Rewrite trajopt_ifopt bindings

The `trajopt_ifopt` module has the biggest API changes. The types formerly in `ifopt::` are now in `trajopt_ifopt::` with redesigned APIs.

**Files:**
- Modify: `src/trajopt_ifopt/trajopt_ifopt_bindings.cpp` (heavy rewrite)
- Modify: `src/tesseract_robotics/trajopt_ifopt/__init__.py` (update __all__)

**Strategy:** Don't rewrite from scratch. Instead, try to build with minimal changes and let the compiler guide each fix. The key changes to make:

**Step 1: Update includes**

Replace:
```cpp
#include <ifopt/variable_set.h>
#include <ifopt/constraint_set.h>
```
With the new trajopt_ifopt core headers (check `ws/install/include/trajopt_ifopt/core/` for exact filenames):
```cpp
#include <trajopt_ifopt/core/variable_set.h>  // or whatever the actual header is
#include <trajopt_ifopt/core/constraint_set.h>
```

**Step 2: Update namespace references**

Replace all `ifopt::` namespace references with `trajopt_ifopt::`:
- `ifopt::VariableSet` → `trajopt_ifopt::Variables` (name changed too!)
- `ifopt::ConstraintSet` → `trajopt_ifopt::ConstraintSet`
- `ifopt::Component` → `trajopt_ifopt::Component`
- `ifopt::Bounds` → `trajopt_ifopt::Bounds`

**Step 3: Update JointPosition to new variable system**

`JointPosition` → `Var`/`Node`/`NodesVariables`. Check what the new constructor looks like in the C++ headers.

**Step 4: Remove cross-module ifopt import**

Remove:
```cpp
nb::module_::import_("tesseract_robotics.ifopt._ifopt");
```

Since ifopt module no longer exists, the base types (Bounds, ConstraintSet, etc.) must be defined in this module directly.

**Step 5: Add ifopt base type bindings**

The base types that were in `ifopt_bindings.cpp` now need to be bound here:
- `Bounds` (with new accessor API: `getLower()`/`getUpper()` instead of `lower_`/`upper_`)
- `Component` (with camelCase methods)
- `Variables` (was `VariableSet`)
- `ConstraintSet` (new `Differentiable` base)
- `CostTerm`
- New: `Differentiable`, `BoundsType`, `RangeBoundHandling`

**Step 6: Update constraint constructors**

All constraints change from `shared_ptr<const JointPosition>` to `shared_ptr<const Var>`.

**Step 7: Remove CollisionCache from evaluator constructors**

`SingleTimestepCollisionEvaluator`, `LVSDiscreteCollisionEvaluator`, `LVSContinuousCollisionEvaluator` no longer take `CollisionCache` as first arg.

**Step 8: Build and fix remaining errors**

```bash
pip install -e . --no-build-isolation 2>&1 | grep "trajopt_ifopt" | head -20
```

Iterate until this module compiles.

**Step 9: Update __init__.py**

Update `src/tesseract_robotics/trajopt_ifopt/__init__.py` `__all__` to:
- Remove `JointPosition`, `CollisionCache` if they no longer exist
- Add new types: `Var`, `Node`, `Variables`, `Bounds`, `ConstraintSet`, `CostTerm`, `Differentiable`, `BoundsType`, `RangeBoundHandling`

**Step 10: Commit**

```bash
git add -u
git commit -m "refactor: rewrite trajopt_ifopt bindings for 0.34 API"
```
